# Santri-Pass: Mobile Monitoring System

**Santri-Pass** adalah purwarupa aplikasi mobile berbasis Flutter yang dirancang khusus untuk mengelola kedisiplinan dan keamanan di lingkungan pondok pesantren. Aplikasi ini mendigitalkan pencatatan izin santri, kunjungan tamu, hingga catatan kesehatan.

---

## 🚀 Fitur Utama (Blueprint)
Berdasarkan Rencana Kerja Aplikasi, berikut adalah fitur utama yang diimplementasikan:
1.  **Digital Permit (Dashboard)**: Monitoring daftar santri yang sedang keluar/masuk secara real-time menggunakan `ListView.builder`.
2.  **Log Kunjungan Tamu**: Pencatatan wali santri/tamu yang berkunjung dengan tampilan kartu yang elegan.
3.  **Titipan & Health Record**: Catatan tambahan untuk manajemen logistik dan kesehatan santri.

---

## 🛠️ Materi Flutter yang Diimplementasikan
Aplikasi ini dibangun dengan menerapkan seluruh materi pertemuan 1 hingga 5:
*   **Pertemuan 1 & 2 (Environment & Dart OOP)**: Penggunaan struktur folder `lib/views`, variabel `List<Map>`, dan implementasi *Constructor* pada Class.
*   **Pertemuan 3 (Basic UI)**: Penggunaan `Scaffold`, `AppBar`, `StatelessWidget`, dan `StatefulWidget`.
*   **Pertemuan 4 (Layouting)**: Implementasi tata letak rapi menggunakan `Column`, `Row`, `Padding`, serta `Expanded` untuk mencegah *overflow*.
*   **Pertemuan 5 (Navigation)**: Manajemen perpindahan layar menggunakan **Named Routes** dan konsep **Stack (Push/Pop)**.

## 💻 Cara Menjalankan Aplikasi
1.  Pastikan Flutter SDK sudah terinstal (cek via `flutter doctor`).
2.  Clone repository ini atau buka folder proyek.
3.  Buka terminal/CMD dan jalankan:
    ```bash
    flutter pub get
    flutter run
    ```
